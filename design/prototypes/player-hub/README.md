# Player Hub interaction prototype

Standalone discussion prototype based only on the approved Player Hub brief and existing champion portrait assets.

Open through the project Vite server at `/design/prototypes/player-hub/`.

Key interactions:

- `/` enters Hunting and focuses roster search.
- Selecting a player enters Reading.
- Selecting a champion enters Digging and updates the realized-gain morph.
- Right-click players or champions for radial commands; releasing inside the dead zone cancels.
- Pinning a second player enters Comparing.
- “Who plays best?” enters Scanning.
- Escape returns to Reading.
- Lock layout and reduce motion are available in the header.

This prototype does not change or connect to production application logic.
